1. Wifite
A free and open-source automated wireless network auditing tool. Wifite uses tools such as Aircrack-ng and Reaver to test WEP and WPA-encrypted wireless networks.
2. Kismet
A free and open-source wireless network detector, sniffer, and IDS. Kismet can run on Windows, macOS, and Linux and tests connections such as Wi-Fi, Bluetooth, Zigbee, and RF
3. Reaver
A free and open-source brute-force attack tool for WPS. Reaver takes between 4 and 10 hours on average to recover a plaintext WPA/WPA2 passphrase.
4. Fern Wi-Fi Cracker
A free and open-source wireless security audit and attack tool for Linux. Fern can help crack and recover WEP/WPA/WPS keys and supports other network-based attacks.
5. Bully
A free and open-source WPS brute-force attack tool. Bully is written in the C programming language and offers improved memory and CPU performance compared with Reaver.
6. CoWPAtty
A free and open-source brute-force WPA2-PSK password cracking tool. CoWPAtty can help users identify weak passphrases that generate the pairwise master key (PMK).
7. InSSIDer
A free Wi-Fi network scanning and troubleshooting tool. InSSIDer provides information about Wi-Fi network configuration and the impact of nearby Wi-Fi networks on performance.
