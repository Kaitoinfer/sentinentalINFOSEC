1. Wireshark
A free and open-source network protocol analyzer and packet capture tool. Wireshark allows users to inspect hundreds of protocols and dozens of file formats.
2. tcpdump
A free and open-source command-line network packet analyzer. Users can specify a particular filter to search for packets that match this description
3. Ettercap
A free and open-source comprehensive suite for man-in-the-middle attacks. Ettercap offers both a command-line and GUI interface and includes features such as live packet sniffing and content filtering.
4. Bettercap
A free and open-source fork of the Ettercap project and so-called “Swiss Army knife” for network attacks. Bettercap can be used on Wi-Fi networks, Bluetooth connections, and 2.4GHz wireless devices
5. Snort
A free and open-source intrusion detection and prevention system. Users can define rules in Snort that indicate malicious network activity and search for packets that match these rules.
6. Ngrep
A free and open-source network packet analyzer that uses grep-like patterns. The ngrep tool supports many different protocols across a wide range of interface types.
7.NetworkMiner
A free and open-source network forensic analysis tool. NetworkMiner can extract files, images, emails, passwords, and more from network traffic in PCAP files.
8. Hping3
A free and open-source command-line packet crafting and analysis tool. The hping3 tool can send custom ICMP/UDP/TCP packets for use cases such as testing firewalls or network performance.
9. Nemesis
A free and open-source packet crafting and injection tool, Nemesis supports many different protocols and can be used for Layer 2 injection on both Windows and Linux systems.
